Z.ai Enhancer is installed

A floating panel that makes chat.z.ai more powerful — sound + visual alerts when the agent finishes, an auto-send queue, and a reusable prompt library.

What you get

  • 🔊
    Sound + toast on completion

    Get an audible chime and a small toast when the agent finishes responding — even if you're in another tab.

  • 📨
    Auto-send queue

    Schedule a sequence of messages. The next one is sent automatically once the agent finishes the previous response.

  • 📚
    Prompt library

    Save reusable prompts with {{variables}}. Click a prompt, fill in the variables, and it goes straight to the chat.

  • 📊
    Response stats

    See the average duration of recent agent responses so you can spot slowdowns.

How to use it

  1. Open chat.z.ai and sign in.
  2. You'll see a circular purple button (FAB) in the bottom-right corner — click it to open the panel.
  3. The panel has 4 sections: Notifications, Prompts, Auto-Send, and Stats.
  4. You can also use keyboard shortcuts (see below).

Keyboard shortcuts

ActionShortcut
Open the panel on chat.z.aiCtrl+Shift+Z
Toggle completion soundAlt+Shift+S
Click the agent Stop buttonAlt+Shift+X

On macOS, use Cmd instead of Ctrl for the first shortcut. Customize at about:addons → ⚙ → Extension Shortcuts.

Privacy

This extension collects no personal data. All your prompts, queue, and stats are stored locally in Firefox and never leave your device. The extension has no analytics, no telemetry, no remote code loading. See the full privacy policy.
